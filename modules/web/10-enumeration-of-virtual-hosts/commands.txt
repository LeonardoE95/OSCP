1 Commands
==========

  Build dokcer image
  ,----
  | docker build -t nginx-lab .
  `----

  we execute it with
  ,----
  | docker compose -f docker-compose.yml up 
  `----

  download wordlist
  ,----
  | wget https://raw.githubusercontent.com/danielmiessler/SecLists/master/Discovery/DNS/subdomains-top1million-5000.txt
  `----

  Using `gobuster'

  ,----
  | gobuster vhost --append-domain --wordlist wordlist/subdomains-top1million-5000.txt -u http://local:1337 
  `----
